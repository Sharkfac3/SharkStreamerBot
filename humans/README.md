# Humans Prompt Starters

This folder is for human-friendly kickoff prompts you can paste into a new chat to activate the right role for a task.

## How to use these

1. Open a new chat.
2. Copy the prompt from the role starter that matches your task.
3. Replace the `Task:` placeholder with your actual request.
4. Send it as your first message.

## Available starters

- `streamerbot-dev-start.md` — for Streamer.bot C# action work under `Actions/`
- `lotat-tech-start.md` — for LotAT engine, schema, and story-pipeline technical work
- `lotat-writer-start.md` — for LotAT stories, lore, worldbuilding, and adventure design
- `brand-steward-start.md` — for public-facing writing, brand voice, canon review, and content strategy
- `art-director-start.md` — for art prompts, character visuals, and stream visual assets
- `app-dev-start.md` — for standalone apps, overlays, dashboards, and stream interaction tooling outside Streamer.bot scripts
- `ops-start.md` — for tooling, validation, sync workflow, and agent infrastructure work

## Notes

- These prompts are meant to front-load the repo/role reading the agent should do before starting work.
- They also remind the agent to check `WORKING.md` first.
- For code-changing tasks, the prompt should push the agent to finish with an ops-style change summary.
- You can create more role starters here over time using the same pattern.
