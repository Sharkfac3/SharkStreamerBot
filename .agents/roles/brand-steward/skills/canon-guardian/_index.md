# Canon Guardian — Overview

## When to Activate This Sub-Skill

- Adding a new character or NPC to the Starship Shamples world
- Extending or modifying existing character personalities
- Creating new ship sections, space regions, or permanent world elements
- Reviewing a generated story for canon consistency
- Proposing new mechanics that interact with existing characters
- Any decision that could affect how existing characters behave or are perceived

## Required Reading

Before any canon-affecting decision:

| Document | Why |
|---|---|
| `Creative/Brand/BRAND-IDENTITY.md` | Metaphor layer — new additions must fit the neurodivergent metaphor |
| `Creative/Brand/CHARACTER-CODEX.md` | Canonical identities — personality, appearance, metaphor role |
| `Creative/WorldBuilding/Agents/D&D-Agent.md` | Established game mechanics and tone |

## The Canon Guardian's Questions

Answer all of these before approving new content:

**1. Does it fit the metaphor?**
- What does this element represent in the neurodivergent/ADHD metaphor?
- Does it add something the current cast doesn't already represent?
- Does it contradict the existing metaphor?

**2. Does it fit the tone?**
- Absurd, chaotic, slightly dramatic, fast-paced, humorous, failure-forward
- Does it introduce darkness that doesn't serve humor?
- Does it work at live stream pace?

**3. Does it conflict with existing characters?**

| Character | Their space |
|---|---|
| Water Wizard | Calm wisdom, hydration solutions, festival brain |
| Captain Stretch | Executive function, authority, shrimp denial |
| The Director | Committee consensus, executive process, four-eyed calm |
| Pedro | Well-meaning escalation, engineering chaos |
| Duck | Hyperfocus rabbit hole, bad ideas that seem fine |
| Clone | Random intrusive thoughts, unexplained presence |
| Toothless | Threat response, security, disproportionate reaction |

A new character must occupy a space none of these already own.

**4. Does it conflict with visual canon?**
- Clashes with the shared anime style?
- Uses a color palette already occupied by an existing character?
- Silhouette reads clearly and differently from existing characters?

**5. Is the cast the right size?**
- Current cast is deliberately lean — 7 characters with distinct roles
- Can an existing character serve the narrative role instead?
- Is this filling a genuine metaphor gap or is it redundant?

## Story Review Checklist

When reviewing a `lotat-writer` story:
- [ ] Each character behaves consistently with their Character Codex personality
- [ ] Pedro causes or worsens at least one problem
- [ ] Captain Stretch's shrimp nature is referenced or hinted at
- [ ] Chaos Meter escalates (does not reset mid-story)
- [ ] Commander moments are rare (1–2 max) and personality-specific
- [ ] Tone stays absurd and fast-paced throughout
- [ ] Endings reflect the final chaos level

## What Requires Escalation to Operator

| Approved without escalation | Requires operator check |
|---|---|
| New space regions (single gimmick, no permanent characters) | New named cast members |
| New mission starting points | Changes to character personalities |
| New Chaos Meter interactions | New permanent ship sections/locations |
| New dice hook flavor text | Mechanics that change how chat interacts |
| Variant expressions of existing characters | Anything requiring CHARACTER-CODEX update |
