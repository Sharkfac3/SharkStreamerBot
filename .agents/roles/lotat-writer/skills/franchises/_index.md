# Franchises — Overview

## What This Folder Covers

Franchise-specific LotAT writing context — recurring settings, canon baselines, tone anchors, and reference docs for a named adventure franchise.

Use this folder when a story idea, adventure draft, or lore note depends on the rules and identity of a specific franchise rather than general LotAT structure.

## When to Load

Load this sub-skill folder when:
- writing or revising story content tied to a specific franchise
- checking franchise-specific canon before expanding adventures or lore
- grounding a new adventure in the established setting, cast, mechanics, or tone of an existing franchise

## Franchise Files

| Franchise | File | Purpose |
|---|---|---|
| Starship Shamples | `starship-shamples.md` | Primary LotAT franchise reference — scope, canon baseline, required docs, and approval boundaries |

## Adding Future Franchises

When a new LotAT franchise is introduced:
- add a new `<franchise-name>.md` file in this folder
- use `starship-shamples.md` as the formatting and scope pattern
- include what the franchise is, its core reference files, canon baseline, and escalation rules
- add the new file to the table above so this folder remains a complete navigation index
