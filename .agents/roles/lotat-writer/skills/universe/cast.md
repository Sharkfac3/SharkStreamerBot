# Cast Canon

## The Rule

The cast is fixed. Do **not** invent new named cast members without explicit operator approval. Each character occupies a distinct personality space — adding characters dilutes existing recognition.

Before proposing a new character: can an existing character serve the narrative role instead?

## Commanders (3 active slots)

| Character | Personality Space | Metaphor Role |
|---|---|---|
| Captain Stretch | Executive function, authority, shrimp denial | The part of the ADHD brain trying to maintain order while things slip |
| The Director | Committee consensus, executive process, four-eyed calm | The deliberate, procedural voice that wants everyone to vote first |
| Water Wizard | Calm wisdom, hydration solutions, festival brain | The grounding presence — always has a water-based solution |

## Squad

| Character | Personality Space | Metaphor Role |
|---|---|---|
| Pedro the Raccoon | Well-meaning escalation, engineering chaos | The good plan that somehow makes everything worse |
| Toothless the Dragon | Threat response, security, disproportionate reaction | The alarm system that treats every problem as a Code Red |
| Duck the Duck | Hyperfocus rabbit hole, bad ideas that seem fine | The intrusive thought that leads somewhere interesting and terrible |
| Clone the Clone Trooper | Random intrusive thoughts, unexplained presence | The thought that just shows up with no context and somehow helps |

## Writing Characters

- Read `Creative/Brand/CHARACTER-CODEX.md` for full personality, appearance, and canonical behavior before writing any character into a scene
- Pedro must cause or worsen at least one problem per story
- Captain Stretch's shrimp nature should be referenced or hinted at
- Commander moments must be personality-specific (not generic heroics)
- All characters must match their metaphor role — do not write them against type
