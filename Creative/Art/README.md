# Art

`Creative/Art/` is for art-generation scaffolding and art-related working materials.

## Belongs here
- reusable art agents
- art prompt templates
- project-specific art work
- approved assets
- visual references and style references

## Does not belong here
- world lore or narrative systems unless primarily art-directed
- tooling scripts
- Streamer.bot runtime source

## Recommended subfolders
- `Agents/` — reusable art-generation scaffolding
- `Projects/` — scoped art work in progress
- `Assets/` — approved/final outputs
- `References/` — source or style references

## Common routing examples
- Character art agent prompt → `Agents/`
- New thumbnail or portrait iteration set → `Projects/`
- Final approved image pack → `Assets/`
