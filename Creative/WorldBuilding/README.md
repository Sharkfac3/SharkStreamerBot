# WorldBuilding

`Creative/WorldBuilding/` is for lore, settings, story systems, and narrative scaffolding.

## Belongs here
- reusable world/story agents
- franchise notes
- settings and lore systems
- storylines and narrative frameworks
- exploratory prompt work tied to world-building

## Does not belong here
- art-only prompt systems
- operational tooling scripts
- Streamer.bot runtime action source

## Recommended subfolders
- `Agents/` — reusable world/story scaffolding
- `Franchises/` — IP or universe-level organization
- `Storylines/` — campaign or narrative arcs
- `Settings/` — places, factions, rules, lore structures
- `Experiments/` — temporary exploratory work

## Common routing examples
- Reusable D&D/world lore agent → `Agents/`
- Setting bible or canon notes → `Settings/`
- exploratory story-agent prompt work → `Experiments/`
