


# What this project does

This project is an AI built system to suppiort sharkfac3 during live streaming. There is already working software we have built in this project that viewers interact with. We also have software like OBS, Streamer.bot, MixItUp, OLLama, and Comfy-UI.

## who is sharkfac3

Sharkfac3 is live streaming on twitch every week night.

Sharkfac3 is building off road racing parts

Sharkfac3 is learning AI

Sharkfac3 is using AI to build software for various content creation pipelines


## live streaming

We live stream to one or two platforms at a time. 

OBS is recording both 16:9 and 9:16 format. 

Streamer.bot is a software that allows C# scripting. 
- actions
- stream-overlay app connection
- MixItUp connection
- Twitch connection via Trigger system

MixItUp is a software that we use to display notifications on OBS source via built in overlay. 
- channel point redeems
- chat messages
- bits
- subscriptions

We have our own app called 'stream-overlay' that we use for mini-games that are live on stream

## meta gaming

we have mini games that 'unlock' squad members.

we have 'commanders' that viewers can redeem to play as on stream.
- unlocks special commands while streaming
- different commanders do different things
